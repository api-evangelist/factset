# FactSet GraphQL API

FactSet provides financial data, analytics, and portfolio management tools. The API covers securities data, pricing, fundamentals, earnings estimates, ownership data, ESG scores, and portfolio analytics for investment management.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from FactSet REST API
**Documentation:** https://developer.factset.com/api-catalog
